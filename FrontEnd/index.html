WadeSafe

Get the info you need to stay safe

Get Started








How to use: First select the day you'd like a report for. Once selected, input the first hour in which the generation schedule changes from 0 in the "First Open-" text-box, then input the last hour before the schedule changes back to 0 into the "Last Open-" text-box. The report will then give the converted time for each access point in the graph below. The output is in military time; between two values is the time where water conditions are not safe to wade. The value on the other side of the bar determines if the time you are looking at is part of a future date.
